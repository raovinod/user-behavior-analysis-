# user-behavior-analysis-
"Analyzing user behavior, cooking preferences, and order trends to derive business insights for improving marketing strategies and customer engagement."


## Project Overview

The datasets used in this project include:
- **UserDetails**: Contains information about the users.
- **CookingSessions**: Includes data on cooking sessions.
- **OrderDetails**: Contains data on the orders placed by users.

The objective of the project is to analyze user preferences, order trends, and behavior based on these datasets.

## Key Findings

- The top 5 popular dishes were identified based on order frequency.
- Orders by gender and age group were analyzed to identify demographic patterns.
- Monthly order trends were visualized to identify peak sales periods.
